# Queue-it-Pow

Queue It POW Solution in JS

## Usage

```javascript
let input = "f02b931c-52f0-4507-9406-f1221678dc16"
let zeroCount = 4;
getHash(input, zeroCount);
```

```python
input = "f02b931c-52f0-4507-9406-f1221678dc16"
zeroCount = 4
getHash(input, zeroCount)
```

```golang
input := "f02b931c-52f0-4507-9406-f1221678dc16"
zeroCount := 4
getHash(input, zeroCount)
```
